D3 Forum

D3 Forum is a duplicable module used to create and manage forums and forum entries.

This module allows you to manage forums but also the comments of any D3 module.

The module features are comparable to the Top PHP forums, discussions and message boards.

D3Forum is also tightly integrated with Trust modules, as it has been around for a long time.

It was developed by Gijoe and Jidaikobo, improved by Naao, Nao-pon and Domifara.

Categories and Forums

Forum Blocks

Adding A Forum Block

Another nice feature you can add blocks with posts, topics and forums to your site provided by the module.

The module creates a block that you can enable to show on the pages of modules you select.

Preferences

Module Preferences focus primarily on how the forums, forum topics, and comments are set on the site.

The D3 Forum module comes with several options set for a new forum.

Review the list of options and define your preferences.

If you are familiar with themes and templates, you can make a copy (duplicate) of the module templates files to customize.

Then, you can edit the files and CSS to meet your theme needs.

Finally, select your Template Set in Render ¬ª Template Management.
